# Gerenciamento de Instituições e Alunos
#### Maven/WEB/SpringBoot com Thymeleaf
#### JPA - Hibernate - MySQL e PostgreSQL
#### Bootstrap e JQuery
#### [Aplicação no Heroku Cloud](https://gia-ucb.herokuapp.com/)

## Features
* CRUD Instituições
* CRUD Alunos
* Pesquisa AJAX de Instituições e Alunos
* Controle de Acesso
